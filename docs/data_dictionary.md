# Data Dictionary

## Overview

The dataset contains Jimmy Award nominees from 2010–2026 and tracks subsequent Broadway career outcomes.

| Variable | Description | Type |
|---|---|---|
| Performer | Performer name | String |
| Year | Jimmy Award nomination year | Integer |
| Winner | Jimmy Award winner status | Binary |
| Gender | Publicly available demographic category | Categorical |
| Broadway_Debut | Whether performer debuted on Broadway | Binary |
| Debut_Year | Year of first Broadway appearance | Integer |
| Timeline_Years | Years between nomination and debut | Continuous |
| Total_Broadway_Credits | Number of Broadway productions | Count |
