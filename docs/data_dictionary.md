# Data Dictionary

## Overview

The dataset contains Jimmy Award nominees from 2010–2026 and tracks subsequent Broadway career outcomes.

| Variable | Description | Type |
|---|---|---|
| Performer | Performer name | String |
| First_Year_Nominated | First Jimmy Award nomination year | Integer |
| Ever_Winner | Whether performer won the Jimmy Award (Y/N) | Binary |
| Gender | Performer gender category used in analysis | Categorical |
| Broadway_Debut | Whether performer has made a Broadway debut (1/0) | Binary |
| Year_Of_Broadway_Debut | Year of first Broadway debut | Integer |
| Timeline_Years | Years between nomination and debut or censoring year | Continuous |
| Total_Broadway_Credits | Number of Broadway productions credited | Count |
